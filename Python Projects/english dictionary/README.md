# English-Dictionary
Given a word, show the meaning of the word - Python for fun

Step 1: Given data.json, parse it to find a matching word with user input and return it's meaning
Step 2: check for upper case and lower case conversions of the word
Step 3: In case of a typo, search for words that match with user query upto 80% and query user if they meant a different word (Kind of like "Did you mean: " feature by Google search)
